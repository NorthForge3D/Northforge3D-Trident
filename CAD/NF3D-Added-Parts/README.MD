# NF3D-Added-Parts

Custom NorthForge3D components designed for the Trident dual-gantry proof-of-concept.

These parts represent motion experiments, toolhead innovations, frame enhancements, bracketry, and other mechanical upgrades that diverge from the baseline Voron Trident footprint.

---

### Expected Contents

- Gantry components  
- Carriage plates  
- Belt routing prototypes  
- Brackets and support structures  
- Electronics mounts  
- Custom hardware adapters  

---

All parts here are subject to rapid iteration and will be reorganized as development stabilizes.
